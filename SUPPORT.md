# Support

- **Question / "how do I…?"** →
  [GitHub Discussions](https://github.com/georgyia/surrounded-by-slop/discussions)
- **Bug** (something behaves wrong) →
  [open a bug report](https://github.com/georgyia/surrounded-by-slop/issues/new/choose)
- **Idea / feature** →
  [open a feature request](https://github.com/georgyia/surrounded-by-slop/issues/new/choose)
- **Security problem** → see [SECURITY.md](SECURITY.md), never a public issue

Issues that are actually questions get converted to Discussions — nothing gets
lost, it just lives where conversations belong.
